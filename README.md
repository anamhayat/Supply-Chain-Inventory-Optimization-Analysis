# Supply-Chain-Inventory-Optimization-Analysis
Situation: Situation: Retailers often face issues like stockouts or excess inventory due to inefficient supply chain planning.

Task:
Analyze supply chain data to identify inventory inefficiencies and improve stock availability.

Action:

Queried supply chain datasets using SQL to analyze inventory levels, shipment times, and demand patterns.

Identified products with frequent stockouts or excess inventory.

Built a dashboard visualizing inventory turnover, lead times, and supplier performance.

Used time-series analysis in Python to evaluate demand trends.

Result:

Highlighted opportunities to improve inventory planning and reduce stockouts.

Demonstrated how better demand forecasting can improve supply chain efficiency and product availability.
